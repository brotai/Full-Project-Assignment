Copyright, Creative Common (CC) Licenses, Fair Use

Works that qualify for copyright protection

  • It must fall within one of the categories or subject matter in which copyright can exist.
  • It must be original.
  • The nationality of the author, or the origin of the work, must be one that qualifies for protection.
  • Certain works must be fixed either in writing or some other material form.

CC BY-SA

  • BY: credit must be given to the creator.
  • SA: Adaptations must be shared under the same terms.

CC BY

  • BY: Credit must be given to the creator.

CC BY-NC

  • BY: Credit must be given to the creator.
  • NC: The material cannot be used for commercial purposes.

CC BY-ND

  • BY: Credit must be given to the creator.
  • ND: No derivatives or adaptations of the work are allowed.

Fair use applies to:

  • Criticism: Commenting on or criticizing the copyrighted work itself.
  • Review: Evaluating the work, such as in a book review or film critique.
  • News Reporting: Using the material to report on current events or issues.
  • Research: Incorporating the material into scholarly or academic research.
  • Private Study: Using the material for personal learning or study.

How to Improve Your Website SEO?

Search engine optimization

Here are five effective strategies to enhance your website's visibility

  • Optimize for Keywords: Use relevant keywords in titles, headings, and content. Conduct keyword research to target high-volume, low-competition terms.
  • Create High-Quality Content: Produce valuable, engaging content and update it regularly to keep it fresh and relevant.
  • Improve User Experience (UX): Ensure your site is mobile-friendly and has fast loading speeds to enhance user experience.
  • Build Quality Backlinks: Acquire reputable backlinks through guest blogging and creating shareable content, avoiding spammy link schemes.
  • Utilize Web Publishing Best Practices: Implement structured data markup, submit XML sitemaps, and optimize meta tags to improve search engine indexing and visibility.

Factors when Choosing a Web Hosting Provider

Web Publishing

When choosing a web hosting provider, consider these factors

  • Performance and Reliability: Ensure the hosting provider offers high uptime guarantees and fast server performance to keep your website accessible and responsive.
  • Scalability: Choose a provider that allows easy upgrades to accommodate growth in traffic and resource needs.
  • Security Features: Look for features such as SSL certificates, firewalls, and regular backups to protect your website from security threats.
  • Customer Support: Opt for a provider with reliable and responsive customer support available through multiple channels like chat, email, or phone.
  • Pricing and Plans: Evaluate the cost of various hosting plans and ensure they align with your budget, while considering the features and resources included.

Web Performance and Maintenance

1: Web Performance

Ways to Reduce Page Loading Time

  • Optimize Images: Compress and resize images to reduce their file size without sacrificing quality.
  • Minimize HTTP Requests: Combine CSS and JavaScript files, and use image sprites to reduce the number of HTTP requests made by the browser.
  • Leverage Browser Caching: Set up caching rules to store static files locally in users' browsers, which speeds up page loading for repeat visitors.
  • Enable Compression: Use Gzip or Brotli compression to reduce the size of your files before sending them to the browser.
  • Use a Content Delivery Network (CDN): Distribute your content across multiple servers globally to decrease load times by serving files from locations closer to users.

2: Web Maintenance

Important Website Maintenance Actions

  • Regular Backups: Create and store backups of your website’s files and database to prevent data loss in case of a failure.
  • Update Software: Keep your website's CMS, plugins, and themes up-to-date to ensure security and compatibility.
  • Monitor Performance: Use tools to regularly check your website’s performance and address any issues that may affect loading times or user experience.
  • Check for Broken Links: Periodically scan your site for broken links and fix or remove them to maintain a good user experience and SEO health.
  • Review Security Measures: Regularly review and update your security protocols, such as firewalls and access controls, to protect your website from vulnerabilities.

Web Security

Common Cyber Attacks and Preventive Measures

  • Phishing: Fraudulent attempts to obtain sensitive information by pretending to be a trustworthy entity. Prevention: Educate users on recognizing phishing attempts, use email filtering solutions, and verify suspicious emails before clicking on links or opening attachments.
  • Malware: Malicious software designed to harm or exploit systems, such as viruses, trojans, and ransomware. Prevention: Install and update antivirus and anti-malware software, regularly scan your system, and avoid downloading or executing unknown files.
  • DDoS (Distributed Denial of Service) Attacks: Overwhelming a network or server with excessive traffic to disrupt service. Prevention: Use DDoS protection services, employ rate limiting, and maintain a robust network infrastructure to handle high traffic volumes.
  • SQL Injection: Exploiting vulnerabilities in a web application to execute malicious SQL commands. Prevention: Use parameterized queries or prepared statements, validate and sanitize user inputs, and regularly test your web applications for vulnerabilities.
  • Man-in-the-Middle (MitM) Attacks: Intercepting and altering communication between two parties without their knowledge. Prevention: Use HTTPS and SSL/TLS encryption for secure communication, implement strong authentication mechanisms, and avoid using unsecured public Wi-Fi for sensitive transactions.
